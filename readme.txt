== Description ==

User to Galette Exporter is a plug-in that synchronise users from Wordpress with Galette users. See http://galette.eu for more explanation about Galette.

== Installation ==

1. Upload User to Galette Exporter directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. The plugin can manage galette users if they are on the same bdd that wordpress (with "galette_" prefix) or on an external bdd (host form in the panel)
